# code-style

My personal code style, packaged for reuse.

This monorepo holds the formatter and linter configurations I use across my projects,
published as installable packages so every project I work on can share the same baseline.

## Packages

- [`@kazukit/oxfmt-config`](./packages/oxfmt-config) — Shared [oxfmt](https://github.com/oxc-project/oxfmt) configuration
- [`@kazukit/oxlint-config`](./packages/oxlint-config) — Shared [oxlint](https://github.com/oxc-project/oxc) configuration
